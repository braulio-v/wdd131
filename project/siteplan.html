Website Planning Document

English Levels Portal – Explore A1, A2, and B1 Content

Site Name

English Levels Portal

This name was chosen because it clearly represents the purpose of the website — helping learners explore and practice English through levels A1, A2, and B1. It is simple, descriptive, and directly communicates what visitors will find on the site.

Optional domain availability: englishlevelsportal.com

Site Purpose

The English Levels Portal website provides structured English learning materials organized by level (A1, A2, and B1). Each level includes lessons with explanations, images, and short videos that focus on grammar, vocabulary, and communication skills. It also offers interactive exercises and quizzes that allow users to practice and track their progress.

  • Grammar and vocabulary lessons for A1–B1 levels
  • Short educational videos and image-based explanations
  • Interactive quizzes to test understanding
  • Level navigation for an easy, personalized experience
  • Resources for both self-learners and English teachers

Scenarios

Example questions visitors might have when visiting the site:

  • Which lessons should I start with if I’m a beginner (A1 level)?
  • How can I practice grammar and vocabulary at my current English level?
  • Are there quizzes to test my progress after each lesson?
  • Can teachers use these materials to help their students?

Color Schema

The selected color palette aims to create a calm, academic, and inviting atmosphere for learners.

Primary: #003366 – Used for headers and navigation
Secondary: #3399cc – Used for buttons and highlights
Accent: #ffd700 – Used for icons and level highlights
Background: #f5f6fa – Used for page background

Typography

Headings Font: Georgia

This serif font will be used for all headings to give the site a professional, academic look.

Body Font: Segoe UI

This modern sans-serif font will be used for body text for clarity and readability.
